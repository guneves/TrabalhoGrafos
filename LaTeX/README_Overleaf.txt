Como usar esta pasta no Overleaf
================================

1. Abra o template WebMedia no Overleaf:
   https://www.overleaf.com/latex/templates/webmedia-template/ckjcbscxjjjh

2. No projeto criado a partir do template, envie/substitua estes itens da
   pasta LaTeX:
   - main.tex
   - artigo_webmedia_referencias.bib
   - a pasta results inteira

3. A pasta results deve conter estes PNGs:
   - graph_visualization.png
   - degree_histogram.png
   - degree_ccdf.png
   - algorithm_times.png
   - degree_log_pk.png
   - degree_model_comparison.png
   - robustness.png
   - degree_distribution.png

4. Defina main.tex como arquivo principal.

5. Compile com pdfLaTeX e BibTeX no Overleaf.
